Ergoemacs mode

Ergonomic anti-RSI keybindings for text editing, based on the US Qwerty version of ergoemacs-mode.

Features

  • Tuned to feel quick and responsive.
  • Does not trigger accidentally while touch typing.
  • Compatible with terminal apps.

Included key bindings

Trigger key + I/J/K/L Arrow keys
Trigger key + U/O Move cursor one word backward/forward
Trigger key + H/; Move cursor to beginning/end of line
Trigger key + D/F Backspace/Delete
Trigger key + E/R Delete one word backward/forward
Trigger key + S/G Delete to beginning/end of line

Usage Notes

  • Space should be your default choice for the trigger key, as it's the easiest to reach and the most ergonomic option. However, Tab is also provided as an option, so that if you have a keyboards with two spacebars, so you can map one to Space, one to Tab, and use both as the trigger key.
  • If you want to use a different keyboard layout or another key as the trigger key, please download the source code and modify this file.

  • In Zsh, the command backward-kill-line does not have a keybinding by default and the shortcut for "Delete to beginning of line" will delete the whole line instead. To get it to work correctly, you need to add the following to your ~/.zshrc file:

    bindkey "^U" backward-kill-line

    If you are using Prezto's editor module, then this is included automatically.
