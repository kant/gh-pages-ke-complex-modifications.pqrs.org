Notes

  • Zsh is the only shell that supports Redo, but unfortunately, it's not bound by default. To use the shortcut listed here, you need to add the following to your ~/.zshrc file: 
    bindkey "^[_" redo
  • If you are using Prezto's editor module, then this is included automatically.
  • This mod supports English and Finnish keyboards. If you want to use a different keyboard layout, please download the source code and modify terminal_undo_redo.json.erb.
